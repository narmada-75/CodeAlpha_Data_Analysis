# Data Analysis Projects

## 1. Titanic Exploratory Data Analysis
- Dataset: Titanic
- Tools: Python, Pandas, Matplotlib, Seaborn
- Insights:
  - Females had higher survival rate
  - First class passengers survived more
  - Age and fare affected survival

## 2. Iris Data Visualization
- Dataset: Iris
- Tools: Python, Pandas, Seaborn, Matplotlib
- Insights:
  - Clear differences between flower species
  - Petal length is key feature
  - Visualization helps pattern detection
