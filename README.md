# Titanic Exploratory Data Analysis (EDA)

## Project Overview
This project analyzes the Titanic dataset to understand survival patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Steps Performed
- Loaded dataset
- Checked missing values
- Performed data cleaning
- Created visualizations
- Analyzed survival patterns

## Key Insights
- Females had higher survival rate than males
- First class passengers survived more
- Age and fare influenced survival chances

## Conclusion
This project helps understand how different factors affected survival on the Titanic.
